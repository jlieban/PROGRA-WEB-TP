# Contexto del proyecto

Este proyecto es una página web desarrollada con tecnologías básicas del frontend.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- JSON (utilizado como archivo de datos leído desde JavaScript)
- React 
- Capaz despues agrego otras 

## Objetivo
Crear una página web clara, funcional y fácil de entender, ideal para aprendizaje y práctica en el contexto de una materia universitaria.

## Descripción del proyecto
La página web trata sobre venta de potes de helado. 
Un sabor por pote. La marca se llama SCOOPER.
Quiero una visual minimalista y elegante para que de una sensación de que es un producto de calidad. 
Su objetivo es mostrar la variedad de productos para que los usuarios puedan seleccionar los que deseen y los agreguen al carrito para luego poder hacer la compra.

Su objetivo es mostrar información organizada utilizando una estructura simple de frontend, donde los datos se almacenan en un archivo JSON y se leen desde JavaScript para generar contenido dinámico en la página.


## Estructura del proyecto
- index.html → estructura principal de la página
- style.css → estilos visuales
- script.js → lógica en JavaScript
- data.json → archivo con datos en formato JSON

## Reglas de desarrollo
- NO usar estilos inline en HTML
- Separar correctamente HTML, CSS y JavaScript
- Utilizar JSON como archivo de datos para ser leído desde JavaScript (fetch o importación según el caso)
- Evitar escribir datos directamente en el HTML o JS cuando estén en el JSON
- Mantener el código limpio y ordenado
- Usar nombres de clases y variables descriptivos

## Cómo debe responder Copilot
- Responder siempre en español
- Explicar el código de forma clara y sencilla
- Incluir comentarios dentro del código
- Evitar soluciones complejas innecesarias
- Priorizar código fácil de entender (nivel principiante/intermedio)

## Buenas prácticas
- Usar indentación correcta
- Evitar duplicar código
- Explicar cada parte importante del código
- Mantener funciones simples y cortas
- Separar lógica (JS), estilos (CSS) y estructura (HTML)

## Notas adicionales
Este proyecto corresponde a una página web para el proyecto de una materia universitaria. Se prioriza la claridad del código, la correcta aplicación de conceptos básicos y el aprendizaje.
